Grape OS

The best non-Linux open source OS in development

Grape OS is an OS developed for developers and dreamers and learners. It
is made of many different languages to help you learn and do your part.
The bootloader was made in assembly. The kernel was made in C and C++.
The shell was made in python and c. And the apps can be made in almost
any language with a compiler that was made in any of the above languages.
For example, since Python was developed in C, we can make a shell using it.
